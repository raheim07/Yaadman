YaadmanLang

A custom programming language built for the Programming Languages Analysis course.

⬡

Lexer

Tokenises source text into a stream of typed tokens: keywords, identifiers, literals, operators.

⬡

Parser

Builds an Abstract Syntax Tree from the token stream using a recursive-descent grammar.

⬡

Semantic Analyzer

Validates scoping, type rules, and bindings before execution to catch logical errors early.

⬡

Interpreter

Tree-walks the AST to execute statements and expressions in a managed runtime environment.
